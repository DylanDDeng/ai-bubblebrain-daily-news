家人们，

快了快了，

感觉离Gemini 3的发布又近了一天。

本来昨天下午在愉快的刷着X，

突然看到一条消息，

![](https://image-1325800846.cos.ap-nanjing.myqcloud.com/20251114005714786.png) 

说Gemini 3 已经可以通过Canvas 在移动端上偷跑了。 

但是确实没人说

这到底是Gemini 3 Pro 还是Gemini 3 Flash。 

我上手体验了一下在Canvas 上的效果， 

虽然不能打包票说这个一定就是Gemini 3， 

但是这效果确实牛逼到爆炸。。。  

![](https://image-1325800846.cos.ap-nanjing.myqcloud.com/20251113071640642.png)

肯定比现在版本的Gemini 2.5 Pro 强很多。

我觉得还是值得分享下我测过的case。 

## SVG   

我先说一下我自己对SVG这类测试的看法：

**它不一定能完全的反映出模型的能力，但是确实是一种比较直观的展现模型能力的方式。**

看一个SVG例子：

``` 
SVG Animation of nintendo switch playing pokemon 
``` 
让它生成一个任天堂switch玩pokenmon游戏的动画。

![](https://image-1325800846.cos.ap-nanjing.myqcloud.com/CleanShot%202025-11-14%20at%2001.25.13.gif) 

坦白说，

这是我所有测过的case里最让我惊艳的一个，

这可太特么牛逼了我觉得，

来对比一下Claude 4.5 Sonnet： 

![](https://image-1325800846.cos.ap-nanjing.myqcloud.com/CleanShot%202025-11-14%20at%2001.31.26.gif) 

还有昨天刚更新的GPT-5.1-Thinking： 

![](https://image-1325800846.cos.ap-nanjing.myqcloud.com/CleanShot%202025-11-14%20at%2001.34.09.gif) 

这特么真的简直不是一个水准的。。。 

还有一个经常测的xbox控制器svg，

测了这么多模型，

真的只有还没发布的Gemini 3做的最好。

```
Draw a svg of xbox controller
``` 
![](https://image-1325800846.cos.ap-nanjing.myqcloud.com/20251114013745685.png)

对比一下Claude 4.5 Sonnet 的效果： 

![](https://image-1325800846.cos.ap-nanjing.myqcloud.com/20251114014506136.png)

我现在强烈怀疑

Gemini 3在画SVG方面应该真是独一档的存在。 

除了SVG， **整体前端美感上应该也是提升了不少**。

## 网页设计  

先看一个最简单的case，让它介绍一下自己：

``` 
Write a html page to introduce yourself including model name, knowledge cutoff date
``` 

![](https://image-1325800846.cos.ap-nanjing.myqcloud.com/20251114015531490.png)

这个最让我觉得牛逼的是，

我其实并没有指定网页的配色，样式要选用什么样的，

但是Gemini 给我直出的效果意外的好。

我们可以用同样的prompt来试验一下Claude的效果，

就明白差别有多少。

## 文学创作   

之前在Web Arena 上测试的时候，

就有小伙伴问我Gemini 3的文字表现能力怎么样。

我在Canvas这里也测试了一下，

先来一个简单点的case： 

``` 
用你的才华写一段足够惊艳我的文字
``` 

这是一个开放命题的要求，

![](https://image-1325800846.cos.ap-nanjing.myqcloud.com/20251114082245166.png) 

再来看看同样文学创作非常不错的Claude。 

![](https://image-1325800846.cos.ap-nanjing.myqcloud.com/20251114082435858.png)

我自己可能还是更喜欢Claude写的， 

相对来说写的内容以及选题稍微好懂一些，

Google这个选题上感觉还是相对抽象一些？ 

再看一个文学翻译的case： 

```
用张爱玲的语言文字风格翻译一下这句话：Love is like a war: easy to begin but very hard to stop. 只需要一个版本，不要过多解释
``` 
![](https://image-1325800846.cos.ap-nanjing.myqcloud.com/20251114084312769.png)

**叠甲说明：这里选择了canvas，但是没触发，所以不确定它一定就是那个效果更牛逼的模型**  

再来看Claude的效果： 

![](https://image-1325800846.cos.ap-nanjing.myqcloud.com/20251114084509690.png)  

我自己是更喜欢Gemini的翻译，

我也让GPT-5.1当了回裁判，

它也认为是Gemini更好。

不过，本身Gemini 2.5写文字这块就不差，

所以具体提升多少

可能也没法一下子很明显的看出来。
