Claude Code 基本已经成为了2025年Agent的一个经典范式。所以我觉得学习如何使用Claude Code 是非常重要的，这篇是这个系列的开始，希望能记录一下如何从零开始使用Claude Code。  

## 准备条件

在使用Claude Code 前，我们需要准备好Claude的订阅账号，或者使用Anthropic 的官方API。 当然，如果你无法使用Claude的话，你也可以使用国内各大模型接入Claude Code的方案，比如智谱，Kimi等等。  

*我们会单独开一篇来讲如何使用Claude Code 接入不同的国内模型。*   

我们后续的记录中还是会使用Claude模型。  

## 安装Claude Code 

安装Claude Code 的方式有很多种， 让我们一一介绍一下：  

**macos, Linux, WSL** 

```
curl -fsSL https://claude.ai/install.sh | bash
```  
如果你使用的是macos，还可以使用Homebrew安装：

```
brew install --cask claude-code
```

**Windows PowerShell** 

```
irm https://claude.ai/install.ps1 | iex
``` 

**Windows CMD** 

```
curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
```  

还可以使用NPM安装， 注意你的Node.js的版本需要在18以上。

```
npm install -g @anthropic-ai/claude-code
```  

安装好之后，就可以直接在你的项目下使用Claude Code 了。 

``` 
cd 你的项目目录 
claude
```  
进入之后，你应该看到的界面跟下面这个差不多：
