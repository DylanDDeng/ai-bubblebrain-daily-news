[上一篇](https://bubblenews.today/my-publish/2026-01-01-notes/)我们讲了下如何安装Claude Code，这篇我们重点来介绍下Claude Code的几个常见的使用命令。  

## 版本查看 

```
claude --version
```  

![](https://image-1325800846.cos.ap-nanjing.myqcloud.com/20260102184848880.png) 

通过这个命令你可以查看你当前的Claude Code 的版本是否是最新的。  

## mcp 

``` 
claude mcp list
``` 

虽然现在Skills 风头更盛，但是mcp 仍然是访问外部服务非常重要的一环。通过上面这个命令你可以查看你的Claude Code接入了哪些mcp服务。 

![](https://image-1325800846.cos.ap-nanjing.myqcloud.com/20260102185614224.png) 

很多mcp服务已经支持一行命令接入Claude Code。比如我最常用的[`chrome-devtools-mcp`](https://github.com/ChromeDevTools/chrome-devtools-mcp).  可以用它来配合Claude Code 配合网页自动化的一些动作，比如网页内容抓取等等。一行命令适配接入Claude Code：  

```
claude mcp add chrome-devtools npx chrome-devtools-mcp@latest
```   

非常建议大家如果要接入mcp的话，阅读每个mcp服务的官方仓库,一般的仓库里都有详细讲解如何配置各个Coding Agent。

## 诊断 

```
claude doctor
```  
通过上面这个命令，可以查看Claude Code 的相关信息 。 

![](https://image-1325800846.cos.ap-nanjing.myqcloud.com/20260102191040368.png) 

## YOLO模式  

这个模式非常危险，建议大家在使用前，先深思熟虑一下。 

```
claude --dangerously-skip-permissions
```
