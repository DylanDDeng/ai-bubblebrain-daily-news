#### 1. CopilotKit 成 GitHub Trending 热门：面向 AI Agent 的前端技术栈，发布 AG-UI 协议

CopilotKit 本周登上 GitHub Trending，这是一个面向 AI Agent 和生成式 UI 的前端技术栈，支持 React 与 Angular。该项目同时发布了 AG-UI 协议，为智能体界面交互提供标准化方案。

- 来源：[GitHub Trending](https://github.com/CopilotKit/CopilotKit)
- 分类：Agent/开发者工具
- AI 评分：7.5

#### 2. 开源工具 Agent-Reach 登上 Trending：AI 智能体的多平台浏览之眼，零 API 费用

Agent-Reach 是一个命令行工具，让 AI 智能体能够阅读和搜索 Twitter、Reddit、YouTube、GitHub、Bilibili 和小红书，全程无需支付 API 费用。该工具登上 GitHub Trending，为开发者构建联网智能体提供了实用方案。

- 来源：[GitHub Trending](https://github.com/Panniantong/Agent-Reach)
- 分类：Agent/开发者工具
- AI 评分：7.5
