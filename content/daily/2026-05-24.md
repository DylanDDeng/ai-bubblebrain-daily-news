>  `AI 日报`

### **Today's AI News**

1. **Understand-Anything** 能够将任意代码转化为可探索、搜索与提问的**交互式知识图谱**，强调教学图表优于炫目图表，并兼容多个主流编码代理工具，已获得 21,877 星。

2. **claude-plugins-official** 是由 **Anthropic** 官方维护的高质量 **Claude Code** 插件目录，星标数已达 26,549。

3. **codegraph** 提供了预建索引的**代码知识图谱**，通过更少的 token 与工具调用实现完全本地化，适用于 Claude Code、Codex 等环境，已获 19,755 星。

4. **ai-engineering-from-scratch** 倡导"学习它、构建它、为他人交付它”的 **AI 工程实践**，获得 13,878 星。

5. **andrej-karpathy-skills** 仅通过一个 **CLAUDE.md** 文件来改善 Claude Code 的行为，灵感来源于 **Andrej Karpathy** 对 LLM 编码陷阱的观察，热度极高，达到 149,843 星。

6. **chrome-devtools-mcp** 将 **Chrome DevTools** 的能力带入编码代理之中，星标数为 41,368。

7. **Anthropic-Cybersecurity-Skills** 提供 754 项结构化**网络安全技能**，映射至 **MITRE ATT&CK**、**NIST CSF** 等 5 大框架，遵循 agentskills.io 标准，兼容 20+ 平台，覆盖 26 个安全领域，获得 7,495 星。

8. **presenton** 是开源的 **AI 演示生成器**与 API，定位为 Gamma、Beautiful AI 等产品的替代方案，获 6,427 星。

9. **multica** 是一个开源的**托管代理平台**，可将编码代理转变为真正的队友，支持任务分配、进度追踪与技能复合，星标数为 31,972。

10. **LongLive** 由 **NVlabs** 推出，专注于长视频生成的 **LongLive 2.0: Infra**，获得 1,834 星。



### **AI每日分析**

**核心趋势总结**  
今天的开源动态揭示了一个清晰转向：AI 编程代理正在从"全能聊天模型”快速进入"可组装技能工具链”时代。社区不再满足于更强的基座模型，而是围绕 Claude Code 等编码代理，疯狂构建知识图谱、插件市场、安全技能库与托管平台，使代理行为更可控、更专业、更像真正的开发队友。

**深度洞察**  
最值得关注的是 **"技能热”背后的接口革命**。`andrej-karpathy-skills` 仅凭一个 `CLAUDE.md` 文件就改变了 Claude Code 的行为，并收获近 15 万星，说明行业正在从沉重的微调与复杂提示工程，转向**轻量、可组合的技能描述层**。这种"声明式代理配置”把人类最佳实践直接注入工作流，比训练更经济、比单纯的 prompt 更稳健。同时，`Anthropic-Cybersecurity-Skills` 提供了 754 项结构化技能并映射到 MITRE ATT&CK 等框架，标志着安全等垂直领域的代理能力正在被**工业化封装**，代理不再"理解安全”，而是直接具备可审计的安全操作单元。这预示着一个"技能市场”的雏形，未来不同代理之间可能通过标准化的技能包互操作，例如 `multica` 已在多代理协作上做出示范。

**一句话前瞻**  
接下来几天，预计会出现连接各代理平台的共享技能注册中心或统一的技能规范草案，推动"代理技能”像软件包一样被分发与复用。
